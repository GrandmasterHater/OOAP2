Пример на Python реализует добавление как общую для всех типов иерархии операцию add у типа Any, 
которую можно переопределить. Сам Vector наследуется от Any и предоставляет свою реализацию 
суммирования. В целом подход простой, понятный и наглядный. 

Пример на Java демонстрирует подход, когда суммирование описывается типом Adder и операцией Sum.
Операция имеет дженерик тип аргументов и результата операции, и может быть применена для любого
типа поскольку не имеет ограничений. Тип Vector наследуется от Adder и реализует операцию Add
в соответствии с требованиями типа. 

В своём решении пошёл несколько иным путём. Определил операцию суммирования как контракт в виде 
интерфейса Addable. Любой, кто должен поддерживать операцию суммирования, реализует этот интерфейс.
Так мы определили единый формат операции применимый к любому типу соответствующему этому контракту.
Далее определил тип Vector как наследник Any поскольку, согласно заданию, использовать этот тип 
можно только для наследников General (для конкретной системы/набора типов). Также реализовал у этого
типа операцию суммирования (реализация интерфейса Addable) и добавил соответсвующее ограничение для 
generic параметра. Написал соответсвующий реализации.
Такой подход позволяет использовать тип Vector для любого наследника General соответствующего контракту
операции сложения.
