### Наследование с ограничением.

#### Пример 1.

   Например есть тип "зелье" описывающий предмет который может применяться ко всем юнитам давая эффекты. Добавив тип "ведьмачье зелье" которое может применяться только к юнитам типа "ведьмак"
используется наследование с ограничением.

#### Пример 2. 

   Типы "конкурс" и "детский конкурс" могут представлять наследование с ограничением т.к. тип "детский конкурс" задаёт дополнительные ограничения для применения типа.

### Наследование с расширением.

#### Пример 1.

   Например тип "словарь" и тип "потокобезопасный словарь" могут быть представлены наследованием с расширением, при условии если функционал словаря дополняется новыми потокобезопасными методами
работы со словарём.

#### Пример 2. 

   Также примером наследования с расширениием может быть иерархия из типов "соединение" и "защищённое соединение" когда тип представляющий интернет соединение дополняется возможностью шифрования.
