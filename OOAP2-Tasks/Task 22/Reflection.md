#### Как понял наследование вида.
   В случае если есть несколько способов классификации и:
   - Каждый из способов важен для предметной области.
   - Способы могут кобинироваться комбинироваться, что имеет существенное влияние на предметную область.
   - Не удаётся выделить приоритетный способ классификации.
   В таком случае каждый вид классификации выводится в собственную иерархию и связывается с целевым типом полиморфно,
связью HAS-A.

#### Эталонные примеры
   Оба примера понятны. Java демонстрирует слуйча когда удалось выделить приоритетный способ классификации, тогда для дополнительного определили 
отдельную ветвь. Пример на Python демонстрирует подход когда оба способа классификации вынесены в собственные ветви иерархии и связаны с целевым
типом композицией. Помимо всего прочего демонстрируется применение других, ранее изученных способа наследования реализации для построения иерархии
классификаций.
